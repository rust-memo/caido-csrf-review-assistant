# دليل CSRF Review Assistant لـ Caido

الإضافة تحلل حركة HTTP داخل Caido بشكل سلبي وتحدد الطلبات التي تحتاج مراجعة CSRF أو CSWSH يدويًا. النتائج ليست ثغرات مؤكدة، والإضافة لا ترسل طلبات اختبار تلقائيًا.

## التثبيت

1. حمّل `plugin_package.zip` من أحدث إصدار في GitHub Releases.
2. افتح **Settings > Plugins** داخل Caido.
3. ثبّت الحزمة وفعّل **CSRF Review Assistant**.
4. ضع النطاق المصرح به داخل Scope، ثم تصفح التطبيق وافتح الإضافة من الشريط الجانبي.

يمكنك أيضًا الضغط بزر الفأرة الأيمن على طلب أو رد محفوظ واختيار **Analyze with CSRF Review Assistant**.

## طريقة المراجعة

1. ابدأ بنتائج **P1 urgent review**.
2. راجع دليل المصادقة والتوكن وOrigin وFetch Metadata وCORS وSameSite وقابلية الطلب للتنفيذ من المتصفح.
3. استخدم **Create Replay variants**. الإضافة تنشئ تبويبات Replay فقط ولا ترسلها.
4. افحص كل نسخة قبل الضغط على Send، خصوصًا طلبات كلمة المرور والبريد والصلاحيات والدفع والحذف.
5. قارن الطلب الأصلي بنسخة حذف التوكن والتوكن الفارغ أو الخاطئ وبيانات المصدر cross-site.
6. لا تعتمد على status code أو تشابه الرد؛ تحقق من تغير الحالة الفعلية داخل الحساب.
7. احفظ حالة المراجعة، ولا تستخدم Confirmed أو Publish Finding إلا بعد تحقق يدوي واضح.

## الأولويات

- **P1:** مصادقة ambient محتملة، وإجراء حساس قابل للتنفيذ من المتصفح، ولا يوجد توكن معروف أو حاجز Cookie قوي ظاهر.
- **P2:** توكن مفقود أو ضعيف مع Method أو Content-Type أو SameSite أو حاجز آخر يحتاج تحققًا.
- **P3:** دليل المصادقة أو تغيير الحالة غير واضح.
- **Protection observed:** ظهر توكن محتمل، لكن التحليل السلبي لا يثبت أن الخادم يتحقق منه.

وجود `Origin` أو `Referer` أو Fetch Metadata أو SameSite لا يثبت أن الحماية مطبقة بصورة صحيحة.

## الأمان والخصوصية

- التحليل السلبي لا يرسل أي اتصال.
- Replay variants لا تعمل إلا إذا ضغط المستخدم Send يدويًا.
- الـPoC محلي، بدون auto-submit، ومتاح فقط لـ GET وURL-encoded POST حتى لا تتغير طريقة الطلب أو أماكن المعاملات.
- التقارير لا تحتوي الطلب الخام أو قيم Cookie وAuthorization والتوكن.
- ملاحظات المراجع مستبعدة من التصدير افتراضيًا.
- لا توجد telemetry أو خدمة AI أو اتصال سحابي.

استخدم حسابات اختبار ونطاقات مصرح بها فقط، وافهم أثر كل طلب قبل إرساله.
